# Open a Carve document

Carve files use the `.crv` extension. Open or create one and the
extension activates automatically, giving you:

- syntax highlighting and semantic tokens,
- diagnostics for parser errors and Djot/Markdown migration warnings,
- context-aware completion (admonition kinds, heading ids, footnote and link
  reference labels),
- hover help, document symbols, folding, rename, and formatting.

Try the bundled `examples/demo.crv` in the repository to see every supported
construct at once.
