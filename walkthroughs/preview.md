# Live preview

Open the preview with the title-bar button, the **Carve: Open Preview**
command, or `ctrl+shift+v` (`cmd+shift+v` on macOS). It opens beside the
editor and updates as you type. The preview:

- renders Mermaid diagrams from ` ```mermaid ` code blocks,
- typesets inline and display math with KaTeX,
- syntax-highlights fenced code blocks,
- follows the active editor, syncs scrolling line-by-line, and highlights the
  block under the cursor.

Configure `@mention` and `#tag` link targets and `:emoji:` glyphs under the
`carve.preview` settings.
