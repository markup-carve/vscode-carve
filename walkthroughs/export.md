# Export your document

- **Carve: Export to HTML** writes a self-contained `.html` file. Mermaid,
  KaTeX, and highlight.js load from a CDN, and theming follows the reader's
  color scheme, so the file renders fully in any browser.
- **Carve: Print Preview / Export PDF** opens the system print dialog on the
  preview - choose "Save as PDF" to produce a PDF.
